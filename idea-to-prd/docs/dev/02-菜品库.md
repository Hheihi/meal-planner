# 模块二：菜品库

## 1. 模块概述

菜品库是核心内容模块，提供按菜系分类的菜品数据管理，支持浏览和搜索。其他模块（推荐、菜单、记录）均依赖菜品库的数据。

## 2. 功能清单

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| D-01 | 菜系分类列表 | P0 | 展示所有菜系分类 |
| D-02 | 菜系下菜品列表 | P0 | 按菜系浏览菜品，分页加载 |
| D-03 | 菜品详情 | P0 | 完整的菜谱、做法、食材、营养信息 |
| D-04 | 菜品搜索 | P0 | 按菜品名称关键词搜索 |
| D-05 | 菜品数据管理（后台） | P1 | 菜品的新增、编辑、上下架 |

## 3. 详细需求

### 3.1 菜系分类 (D-01)

**预设菜系**：

| 菜系 | 说明 |
|------|------|
| 川菜 | 麻辣为主，如麻婆豆腐、水煮肉片 |
| 粤菜 | 清淡鲜美，如白切鸡、蒸鱼 |
| 湘菜 | 香辣，如剁椒鱼头、小炒肉 |
| 鲁菜 | 咸鲜，如糖醋鲤鱼、九转大肠 |
| 江浙菜 | 甜鲜，如东坡肉、西湖醋鱼 |
| 西餐 | 牛排、意面、沙拉等 |
| 日韩料理 | 寿司、拉面、泡菜等 |
| 家常菜 | 不分菜系的日常菜品 |
| 汤粥 | 各类汤品和粥品 |
| 面食 | 面条、饺子、包子等主食类 |

**业务规则**：
- 菜系列表由后台配置，支持排序
- 每个菜系至少包含 20 道菜品（V1 验收标准）
- 菜系图标使用统一风格

### 3.2 菜品列表 (D-02)

**展示信息**：
- 菜品主图（缩略图）
- 菜品名称
- 烹饪时间
- 难度标识（简单/中等/复杂）
- 热量值

**业务规则**：
- 默认按后台排序权重排列
- 分页加载，每页 20 条
- 仅展示状态为"上架"的菜品
- 支持下拉刷新和上拉加载更多

### 3.3 菜品详情 (D-03)

**页面内容**：


**"加入菜单"按钮**：

- 加入成功后显示 Toast 提示"已加入午餐菜单"
- 如果该菜品已在对应餐次中，提示"该菜品已在午餐菜单中"

### 3.4 菜品搜索 (D-04)

**搜索规则**：
- 搜索范围：菜品名称
- 支持模糊匹配（如搜"豆腐"匹配"麻婆豆腐"、"家常豆腐"等）
- 搜索结果按相关度排序
- 无结果时显示"未找到相关菜品，换个关键词试试"

**交互**：
- 搜索页顶部为搜索输入框，支持实时搜索（防抖 300ms）
- 输入框下方展示搜索结果列表，样式同菜品列表
- 搜索框为空时显示热门搜索关键词（可选）

## 5. API 接口

### 5.1 获取菜系列表

```
GET /v1/cuisines
```

**响应**：
```json
{
  "code": 0,
  "data": {
    "list": [
      {
        "id": 1,
        "name": "川菜",
        "icon_url": "https://...",
        "dish_count": 35
      }
    ]
  }
}
```

### 5.2 获取菜系下菜品列表

```
GET /v1/cuisines/:cuisine_id/dishes?page=1&page_size=20
```

**响应**：

```json
{
  "code": 0,
  "data": {

}
```

## 6. 数据要求

### 6.1 初始数据规模

- 每个菜系至少 20 道菜品
- 10 个菜系 × 20 道 = 至少 200 道菜品
- 每道菜品需完整的食材清单和制作步骤

### 6.2 菜品数据质量标准

| 字段 | 要求 |
|------|------|
| 菜品名称 | 不超过 20 字，准确描述菜品 |
| 菜品图片 | 清晰，尺寸不小于 750×500px，JPG/PNG 格式 |
| 食材清单 | 列出所有主要食材和用量 |
| 制作步骤 | 至少 3 步，每步配图可选但建议有 |
| 营养信息 | 基于标准食物营养数据库估算，标注"仅供参考" |

### 6.3 图片规范

- 菜品主图：750×500px，用于详情页
- 菜品缩略图：300×200px，用于列表页
- 步骤图：750×auto，用于制作步骤

## 7. 边界与异常处理

| 场景 | 处理方式 |
|------|---------|
| 菜系下无菜品 | 显示空状态"暂无菜品，敬请期待" |
| 菜品图片加载失败 | 显示默认占位图 |
| 搜索无结果 | 提示"未找到相关菜品，换个关键词试试" |
| 网络异常导致列表加载失败 | 显示错误提示和重试按钮 |
| 菜品被下架后用户访问详情 | 提示"该菜品已下架" |
