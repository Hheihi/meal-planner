## 使用方式

开发某个模块时，将以下内容一起提供给 Claude：

1. **对应的 Plan 文档**（如 `plan-09-今日菜单页.md`）
2. **关联的需求文档**（Plan 中"关联文档"列出的模块文档）
3. **前端架构设计**（`08-前端架构设计.md`）
4. **前端开发规范**（`09-前端开发规范.md`）

这样可以确保生成的代码符合项目架构和规范。
